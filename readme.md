# 什么是开源项目
## 开源项目概念
开源项目（Open Source Project） 就是把软件或者项目的源代码完全公开，任何人都可以查看、使用、修改、分发的项目。
## 开源项目特点
### 源代码公开可见
项目代码托管在公开平台（GitHub、Gitee、GitLab 等），所有人都能阅读源码，知道程序是怎么实现的。
### 社区协作模式
不只是作者一个人维护，全球开发者都可以：
- 提交 Issue 反馈 Bug
- 提交 PR 贡献代码
- 参与讨论、完善文档
### 免费且可二次开发
大多数开源项目可免费使用、二次封装、集成到自己的产品里，极大降低开发成本。
# 什么是Google studio
## Google studio概念
Google Studio 通常指 Google AI Studio，是谷歌官方推出的 云端生成式 AI 开发平台（Web IDE），核心是让你用浏览器快速实验、开发并部署基于 Gemini 大模型的 AI 应用。
# vibe coding 与智能体的介绍
## vibe coding的概念
核心是用自然语言描述需求（氛围 / 感觉），由 AI 全自动生成、调试与维护代码，开发者从编码者转为引导者/验收者
## 智能体的概念
以大语言模型为大脑，具备感知、规划、记忆、工具调用、自主执行能力，能主动完成复杂目标的智能系统。不是只会聊天的机器人，而是能自己思考、自己干活、自己解决问题的数字员工。
# GitHub.io主页是什么
